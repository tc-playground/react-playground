# VSCode Extensions

## Steps

1. Install the `Yeoman` scaffolding tool globally, and the `generator-code` plugin:

    ```bash
    npm install -g yo
    npm install -g yo generator-code
    ```

2. Generate a new plugin:

    ```bash
    yo code
    ```
